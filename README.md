# AffinityAnswers Internship Assessment

This repository contains solutions to all three assessment questions for the AffinityAnswers internship application.

## Repository Structure

### 1. [OLX Web Scraper](./1-olx-web-scraper)
Python script to extract car cover listings from OLX with smart filtering.

**Tech Stack**: Python, Selenium, BeautifulSoup4

### 2. [Rfam SQL Queries](./2-rfam-sql-queries)
SQL queries analyzing the Rfam database for taxonomy and DNA sequence data.

**Tech Stack**: MySQL

### 3. [India Mutual Fund NAV Extractor](./3-india-mutual-fund-nav-extractor)
Shell script to download and extract mutual fund NAV data from AMFI.

**Tech Stack**: Bash, AWK, cURL
